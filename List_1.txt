1. Quem desenvolveu a máquina analítica?

A Máquina Analítica foi desenvolvida por Babbage em 1823. O objetivo do trabalho de Babbage era construir uma máquina programável
As características da Máquina Analítica incluíam:
• Possuía um programa que podia mudar o funcionamento da máquina.
• Era de uso geral.
• Tinha memória, processador, controle e saída.
• Acrescentava a capacidade de desvio (JUMP).
• Os programas eram escritos por Ada Lovelace.

6. Quais eram as características básicas da arquitetura proposta por John von Neumann?

As características básicas da arquitetura proposta por John von Neumann, desenvolvida no projeto de construção do computador IAS em 1946, definem o que hoje conhecemos como o modelo fundamental de um computador moderno.
As principais características incluem:

• Programa Armazenado: O grande diferencial e insight que separa o computador moderno das grandes calculadoras da época é o conceito de instruções de programa e dados convivendo em um mesmo espaço de memória e armazenamento. Diferente da Máquina de Babbage, onde programas eram armazenados em cartões perfurados separadamente dos dados, na arquitetura de Von Neumann, ambos compartilham o mesmo espaço.

• Componentes Básicos de um Computador Eletrônico Digital Moderno: A arquitetura de Von Neumann especifica os seguintes elementos:
    ◦ Uma unidade de processamento (CPU) que contém uma unidade lógica aritmética (ULA) e registradores de processo.
    ◦ Uma unidade de controle (UC) que contém o registrador de instrução e o contador de programa.
    ◦ Uma unidade de memória que pode armazenar tanto dados quanto instruções.
    ◦ Armazenamento externo.
    ◦ Mecanismos de entrada e saída (E/S).

Essa descrição abrange o que hoje chamamos de CPU, RAM e outros componentes, sendo a base da arquitetura de hardware de todo computador moderno.

The Analytical Engine was developed by Babbage in 1823. The goal of Babbage's work was to build a programmable machine.
The features of the Analytical Engine included:
• It had a program that could change the machine's operation.

The purpose behind the development of the world's first electronic computer, the Colossus, built by Alan Turing in 1943, was to break German cryptography.
Specifically, the Colossus was a machine designed to break the Lorenz machine codes. This effort was an important step in helping end World War II in favor of the Allies.

6. What were the basic features of the architecture proposed by John von Neumann?

The basic features of the architecture proposed by John von Neumann, developed in the IAS computer construction project in 1946, define what we now know as the fundamental model of a modern computer. The main features include:

• Stored Program: The major difference and insight that separates the modern computer from the large calculators of the time is the concept of program instructions and data coexisting in the same memory and storage space. Unlike Babbage's Machine, where programs were stored on punched cards separately from the data, in the Von Neumann architecture, both share the same space.

• Basic Components of a Modern Digital Electronic Computer: The Von Neumann architecture specifies the following elements:
◦ A processing unit (CPU) containing an arithmetic logic unit (ALU) and process registers.
◦ A control unit (CU) containing the instruction register and the program counter.
◦ A memory unit that can store both data and instructions.
◦ External storage.
◦ Input/output (I/O) mechanisms.

This description covers what we now call the CPU, RAM, and other components, which are the basis of the hardware architecture of every modern computer.
